# Setup Environment

1. Clone this repo.

   ```terminal
   git clone https://github.com/Nivilian/Rapidly-Exploring-Random-Trees.git
   ```


2. Install the required libraries.

  ```terminal
  pip install -r requirements/requirements.txt
  ```
